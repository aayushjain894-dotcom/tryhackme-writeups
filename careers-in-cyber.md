# Careers in Cyber - TryHackMe

**Date:** June 17, 2026
**Platform:** TryHackMe
**Path:** Pre-Security / Beginner

## What I Learned
Four core cybersecurity roles and how they connect:

- **SOC Analyst** — detects attacks in real time via log monitoring
- **Penetration Tester** — simulates attacks to find how a breach occurred
- **Forensic Analyst** — investigates what an attacker did after gaining access
- **Security Engineer** — implements fixes to prevent future attacks

## Reflection
Each role represents a different phase of the attack lifecycle.
SOC = detect. Pentester = simulate. Forensic = investigate.
Engineer = prevent. Understanding all four helps me see where
I want to specialize long term.
