# Search Skills - TryHackMe

**Date:** June 17, 2026
**Platform:** TryHackMe
**Path:** Jr Penetration Tester / Beginner

## What I Learned

- **Shodan** — search engine for internet-connected devices; 
  reveals open ports and exposed services
- **VirusTotal** — aggregates antivirus engines to analyze 
  files and URLs for malware
- **CVE Database** — catalogs known vulnerabilities with a 
  severity score (CVSS) to prioritize threats
- **Technical Documentation** — official docs are primary 
  source for understanding how tools and protocols work
- **GitHub** — useful for finding Proof of Concept (PoC) 
  exploit code; also exposes flawed repos with vulnerabilities

## Reflection
OSINT tools like Shodan and VirusTotal are used by both 
attackers and defenders. Knowing where to search is half 
the skill in cybersecurity.
