# Defensive Security Intro - TryHackMe

**Date:** June 15, 2026
**Platform:** TryHackMe
**Path:** Pre-Security / Beginner

## What I Learned
- Defensive security focuses on detecting and stopping attacks
- Analyzed logs to identify suspicious activity from a malicious IP
- Identified hidden pages the attacker was attempting to access
- Blocked the attacker by adding their IP to the firewall

## Tools/Concepts Introduced
- Log analysis
- IP tracking
- Firewall rules

## Reflection
Offensive security is about breaking in — defensive is about 
detecting and responding. Both perspectives are needed to 
understand the full picture. SOC analysts do this work daily.
